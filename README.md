# FUTURE_DS_01

Business Sales Performance Analytics
Project Code: DS
Task Number: 01

📊 Project Overview
This project analyzes the Superstore Sales dataset to identify key revenue drivers and regional performance.

🛠️ Tools Used
Microsoft Excel: Pivot Tables, Slicers, and Data Visualization.

💡 Key Insights
Top Category: Technology generates the highest sales.

Top Region: The West region is the most profitable.

Growth Trend: Sales peak during the end-of-year holiday season.
